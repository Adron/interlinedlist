## Platform Overview

InterlinedList is a time-series based micro-blogging platform with interactive list capabilities. Key features include:

- Micro-blogging similar to Mastodon
- Interactive lists with DSL scripts
- User profiles and social features
- OAuth authentication support
- Privacy controls
