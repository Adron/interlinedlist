# Using InterlinedList Platform

This guide covers the main features and how to use them once the platform is fully operational.

## Creating Posts

### Basic Post Creation

1. Navigate to the compose page
2. Write your post content (character limit applies)
3. Add any attachments or media
4. Set visibility (public, followers only, etc.)
5. Click "Post" to publish

### Post Features

- **Time-series ordering**: Posts appear in chronological order
- **Media attachments**: Add images, videos, or other media
- **Visibility settings**: Control who can see your posts
- **Editing**: Edit your posts within a time window
- **Deleting**: Remove posts you no longer want visible

## Creating Interactive Lists

### What are Interactive Lists?

Interactive lists are special posts that use embedded DSL (Domain-Specific Language) scripts to create dynamic, interactive content.

### Creating a List

1. Navigate to "Create List"
2. Enter a title and description
3. Write your DSL script in the editor
4. Preview your list
5. Publish when ready

### List Features

- **DSL Scripts**: Create interactive elements using the platform's DSL
- **Dynamic Content**: Lists can update based on user interactions
- **Sharing**: Share lists with others
- **Embedding**: Embed lists in posts or other content

## Following and Followers

### Following Users

1. Visit a user's profile
2. Click the "Follow" button
3. You'll receive updates from users you follow

### Managing Followers

- View your followers list
- Approve or deny follow requests (if enabled)
- Block users if necessary

## Notifications

### Notification Types

- New followers
- Mentions in posts
- Replies to your posts
- List interactions
- System updates

### Managing Notifications

- View all notifications in the notifications panel
- Mark notifications as read
- Adjust notification preferences in settings

## Search and Discovery

### Searching

- Search for users by username
- Search for posts by keywords
- Search for lists by title or tags

### Discovery Features

- Explore trending posts
- Browse popular lists
- Discover new users
- View recommended content

## Privacy Settings

### Account Privacy

- Make your account private (approve followers)
- Control who can see your posts
- Manage blocked users
- Control data visibility

### Content Privacy

- Set default post visibility
- Control who can mention you
- Manage who can reply to your posts

## Tips for Best Experience

1. **Complete your profile**: Add a bio and avatar to help others find you
2. **Use hashtags**: Make your posts discoverable
3. **Engage with others**: Reply, like, and share content
4. **Create quality lists**: Well-crafted interactive lists get more engagement
5. **Follow community guidelines**: Keep the platform welcoming for everyone
