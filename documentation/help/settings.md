# Settings

Access your settings from the user dropdown (top right) → **Settings**.

## Profile Settings

- **Display Name**: How you appear to others (defaults to username if not set)
- **Bio**: Short description for your profile
- **Avatar**: Profile picture; upload or set from a URL

## View Preferences

- **Messages per page**: How many messages to load at once (10–30)
- **Viewing preference**: Show My Messages, All Messages, Followers Only, or Following Only
- **Show link previews**: Toggle link preview cards on or off

## Message Settings

- **Default message visibility**: Set whether new messages default to public or private
- **Max message length**: Character limit per message (default 666)
- **Advanced post settings**: Show advanced options when composing

## Profile Location

- **Latitude / Longitude**: Optional location for your profile (used by widgets like Weather)

## Permissions

- **Email visibility**: Control who can see your email
- **Account visibility**: Make your account private (requires approval for new followers)
