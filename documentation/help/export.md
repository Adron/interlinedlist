# Exporting Data

Export your data from the **Exports** page (accessible from the dashboard or navigation).

## What You Can Export

- **Messages**: All messages you've posted

- **Lists**: Your list definitions and schemas

- **List Data Rows**: The actual data in your lists

- **Follows**: Your followers and following relationships

## Export Formats

Exports are typically provided as **CSV** (comma-separated values) files. Download the file and open it in a spreadsheet or text editor.

## How to Export

1. Go to the **Exports** page
2. Choose the data type you want to export
3. Click the export button
4. The file will download automatically

## Use Cases

- **Backup**: Keep a local copy of your data
- **Migration**: Move data to another system
- **Analysis**: Analyze your messages or list data in other tools
- **Reporting**: Create reports from your exported data
